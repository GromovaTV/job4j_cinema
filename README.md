# Cinema

Service for buying tickets to the cinema. You can track
and book free seats by purchasing a ticket.

### Dependencies

* Java
* Servlets
* HTML, JavaScript(JQuery, Ajax), Bootstrap
* JDBC + PostgreSQL
* Tomcat
* Log4j, Slf4j
* Maven
* Checkstyle
